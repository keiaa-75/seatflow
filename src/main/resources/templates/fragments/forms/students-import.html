CSV Format Guide

Your CSV file should have the following columns in order:

  1. Student ID (required)
  2. First Name (required)
  3. Middle Name (optional)
  4. Last Name (required)
  5. Suffix (optional)

Tip: Make sure your CSV file uses commas as separators and includes a header row.
